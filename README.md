## 项目概要

- 项目初衷 用来看一些仅限港澳台地区的 [动漫] [ACG] [Cover]  
- 本项目用到的技术栈： vue-cli + vue-router + vuex + axios + vue-axios + vant( UI ) 

## 主要功能

1. 登录，
2. 注册
3. 账号个人中心
4. 保持登录状态 		
5. 多少天免登陆
6. 用户背景板
7. 渲染视频列表
8. 视频下方评论     
9. 第三方登录
10. 按赞
